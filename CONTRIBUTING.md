# Contributing to Rogue Forge

Thank you for your interest in contributing to Rogue Forge.

We welcome contributions of all kinds, including:

- Bug reports
- Feature suggestions
- Documentation improvements
- Code contributions
- Performance optimizations

## Getting Started

1. Fork the repository.
2. Create a new branch.
3. Make your changes.
4. Submit a Pull Request.

Please keep pull requests focused and well documented.

## Code Style

- Write clean, readable code.
- Comment complex logic.
- Keep commits descriptive.

Thank you for helping improve Rogue Forge.
